## 📻 إذاعات الأردن

[![version](https://img.shields.io/badge/version-2.0.0-green.svg)](https://github.com/iofahmawi/jo-radios)
[![PWA](https://img.shields.io/badge/PWA-Enabled-brightgreen)](https://iofahmawi.github.io/jo-radios/index.html)

مشغل ويب بسيط، أنيق، وقابل للتخصيص بالكامل للاستماع إلى محطات الراديو الأردنية مباشرة. تم بناؤه باستخدام HTML و CSS و JavaScript فقط، دون الحاجة إلى أي إطارات عمل معقدة.

### 🔗 تجربة التطبيق
* [🚀 معاينة حية للتطبيق](https://iofahmawi.github.io/jo-radios/index.html)

### ✨ الميزات الرئيسية

* **تشغيل فوري**: قائمة مجهزة بأشهر الإذاعات الأردنية جاهزة للتشغيل.
* **تخصيص كامل للواجهة**: تغيير عنوان الصفحة، إضافة شعار مخصص بأبعاد قابلة للتعديل، والتحكم في حجم الخطوط.
* **إدارة متقدمة للقنوات**: إضافة، تعديل، حذف، وإعادة ترتيب المحطات بسهولة عبر السحب والإفلات.
* **الاستيراد والتصدير**: تصدير واستيراد قائمة القنوات أو ملف شخصي كامل يشمل جميع إعدادات الواجهة.
* **دعم الوضع الليلي والنهاري**: تصميم مريح للعين في جميع ظروف الإضاءة.
* **متعدد اللغات**: واجهة تدعم اللغة العربية والإنجليزية بشكل كامل.
* **تطبيق ويب تقدمي (PWA)**: يمكن تثبيت الموقع على جهازك (كمبيوتر أو هاتف) كتطبيق مستقل.
* **حفظ محلي**: يتم حفظ جميع تخصيصاتك وإعداداتك في متصفحك تلقائيًا.

### 🛠 التقنيات المستخدمة

* **HTML5 & CSS3**: مع استخدام المتغيرات (Variables) لتسهيل تخصيص المظهر.
* **JavaScript (Vanilla ES6+)**: لا يوجد إطارات عمل خارجية.
* **Plyr.io**: لمشغل الصوت الأنيق.
* **HLS.js**: لدعم تشغيل بث HLS (.m3u8).
* **SortableJS**: لتفعيل ميزة السحب والإفلات لترتيب القنوات.
* **Phosphor Icons**: لمجموعة الأيقونات المستخدمة.

### 🚀 كيفية الاستخدام

1. **فتح التطبيق**: اضغط على رابط التجربة أعلاه.
2. **الاستماع**: اضغط على أي إذاعة لبدء الاستماع فوراً.
3. **التخصيص**: اضغط على أيقونة الترس `⚙️` في الزاوية العلوية للوصول إلى الإعدادات.
4. **الإعدادات المتقدمة**: من الإعدادات يمكنك تغيير المظهر، اللغة، إدارة القنوات، وتخصيص الواجهة بالكامل.
